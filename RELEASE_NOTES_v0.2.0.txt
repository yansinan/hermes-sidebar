Release v0.2.0 - Major Architectural Refactoring
===================================================

Date: May 18, 2026

OVERVIEW
--------

v0.2.0 introduces comprehensive codebase reorganization with semantic layering of shared modules, significantly improving code organization and maintainability while maintaining full backward compatibility.

MAJOR CHANGES
-------------

1. SEMANTIC LAYERING ARCHITECTURE

   src/shared/types/      - Type definitions and contracts (8 files)
   src/shared/utils/      - Utility functions and helpers (3 files)
   src/shared/domain/     - Business logic and domain models (2 files)
   src/shared/            - Core extraction and conversion modules

2. COMPONENT DIRECTORY RESTRUCTURING

   src/sidepanel/components/ reorganized from flat to semantic:
   - composer/         - Draft message composition
   - conversation/     - Message display and thread UI
   - overlay/          - Modal dialogs and drawers
   - shared/           - Reusable UI primitives
   - topbar/           - Extension header and status

3. RUNTIME MODULE IMPROVEMENTS

   - New src/runtime/index.ts barrel export
   - New src/runtime/selection-summary-action.ts for selection-based summaries
   - Improved function signatures with explicit template parameters
